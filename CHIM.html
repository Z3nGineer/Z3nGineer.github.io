CHIM Patches

Original work. Where the dream is reshaped by hand.

Eternal Darkness — Nighttime & Interior Tuning

Outdoor skylight hidden via SetActorHiddenInGame for true darkness. Moonlight intensity raised to 0.7 for navigable nights. Interior skylight hiding disabled — brightness controlled by UltraPlusExtensions skylight profile instead. Generic interior point lights left at vanilla. Spotlight brightness tuned: caves 0.85, other 0.75.

UltraPlusExtensions — Interior Skylight Profile

Custom-tuned skylight_ultraplus_choice_brighter_v2.ini with per-location multipliers. Caves, ruins, mines at 0.35. Dungeons at 0.4. Forts, sewers at 0.45. Houses, inns at 0.6. Exteriors unchanged at 1.25. Balances atmosphere with playability.

LumenRemastered & ShadersRevised — API Compatibility Fix

Both mods call ExecuteConsoleCommand with 3 parameters, but UE4SS now requires 4. CHIM patches both mods to pass playerController as the 4th argument, preventing crashes on level transitions and fast travel.

Controller Force Feedback Fix

Sustained spell effects trigger permanent controller vibration that never stops. CHIM adds bEnableForceFeedback=False to Engine.ini, disabling force feedback for Oblivion only. Other games unaffected.

Curated Plugins.txt Load Order

Hand-ordered plugin load list ensuring all ESPs load in the correct sequence. Base game, DLCs, unofficial patch, then mods organized by category. Prevents conflicts and ensures overrides work correctly.
