Original Work

CHIM Patches

Where I had to write custom code to make things work together. These are the patches that come with the modpack.

Eternal Darkness — Night & Interior Tuning

Outdoor skylight hidden for real darkness, moonlight bumped to 0.7 so you can actually navigate. Interior skylight hiding disabled — brightness is handled by the UltraPlus profile instead. Generic interior lights left at vanilla so dungeons aren't pitch black.

UltraPlusExtensions — Interior Brightness Profile

Custom-tuned skylight_ultraplus_choice_brighter_v2.ini with per-location multipliers. Caves and ruins at 0.35, dungeons at 0.4, houses and inns at 0.6. Dark enough to feel atmospheric, bright enough to see what's trying to kill you.

LumenRemastered & ShadersRevised — API Fix

Both mods call ExecuteConsoleCommand with 3 params but UE4SS now needs 4. I patched both to pass playerController as the 4th arg so they stop crashing on fast travel and level transitions.

Controller Force Feedback Fix

Sustained spell effects trigger permanent controller vibration that never stops. Added bEnableForceFeedback=False to Engine.ini. Only affects Oblivion — other games keep rumble.

Curated Load Order

Hand-ordered Plugins.txt with every ESP in the right sequence. Base game, DLCs, unofficial patch, then mods by category. Tested for conflicts so you don't have to.
